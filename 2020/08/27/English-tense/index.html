英语学习——时态

发表于 2020-08-27 更新于 2020-08-28 阅读次数：
本文字数： 3.6k 阅读时长 ≈ 3 分钟

想来自己英语太差劲了，去youbute上随便搜了搜，发现这个老师讲的还不错。如果能娶到这么好的老师当老婆。啊，不是。如果当初有这么好的老师来教英语，也不至于学成一团糟。

老师讲的比较系统，初高中都是做题看到一个讲一个，到时候学一个记一个，很乱。我整理了老师讲课的笔记之类，做了归纳整理。一是这是我自己的学习笔记，二是万一方便后来者呢？

现在进行时

表示正在进行的动作（having breakfast）；表示即将发生的动作（He’s dying, I’m moving this weekend）；

  • 肯定句：subject + (am, is, are) + doing.
  • 否定句：subject + be + not + doing.
  • 疑问句：be + subject + doing?
  • 特殊疑问句：疑问词 + be + subject + doing?

一般过去时

时间与谓语动词的形态。和表示过去的时间状语连用，yesterday, last week, two years ago, in 1999, since（主句现在完成，从句一般过去式）。

使用场合：

  • 过去事件发生的动作或存在的状态，动作（其他动词）和状态（be）区分开；
  • 过去某个时间段的习惯，反复发生的动作（我奶奶过去经常给我讲故事）；
  • 讲到已经死去的人。

谓语动词结构

  • 一般动词：subject + 动词过去时，subject + didn’t + 动词原形, Did + subject + 动词原形？（均借助助动词）
  • be动词（没有动词，表达状态，如他昨天在那）：subject + was/were（wasn’t, weren’t） …

现在完成时

使用场合：

  • 表示动作一致持续到现在。I have lived in Beijing for ten years. So far, there has been no news of him.
  • 动作过去发生，发生时间不明，影响持续到现在。I have just lost my key. I lost the key this morning.（有具体时间点） ，I had breakfast at 8 o’clock.

标志词：just, lately, recently, yet（还….，否定句，句子末尾）, already（肯定句，句子中间）, ever（曾经）, never, so far, up to now, before（之后没有东西）, for …, sicnce…。

不可延续性动词不能和表示一段时间的状语连用。die, marry（嫁）, come, arrive, leave, go, finish, buy。借助 be 动词，My dog has been dead for three years. He has been in Beijing for three years.

谓语动词结构

  1. 肯定句：subject + have/has（表示现在） + done（动词过去分词，表示完成）
  2. 否定句：subject + haven’t/hasn’t + done
  3. 疑问句：have/has + subject + done

过去完成时

使用条件：表示动作发生在过去某时之前，强调过去的过去。

  • 肯定句：subject + had + done （had 是体现时态的助动词，表示过去）
  • 否定句：subject + had not /hadn’t + done
  • 疑问句：had + subject + done?

Before I got home, the rain had stopped.

After I had turned off the light, I left the room. = After I turned off the light, I left the room.（也可以用一般过去时）

一般将来时

表示将来发生的动作。shall 只用于第一人称。

  • 肯定句：subject + will / shall + do
  • 否定句：subject + will/shall + not + do (won’t, shan’t)
  • 疑问句：will/shall + subject +do

将来时态及结构：

  • will + do
  • be + doing 表示即将发生的动作
  • be + going + to + do 表示打算或者即将发生的事
  • mean to do, plan to do, intend to do, hope to do. 打算的意思，用于将来时
  • be to do，表示即将发生的动作
  • be about to do，表示马上发生的动作
  • will be doing 将来进行时，表示意料之中将会发生的事

将来进行时

结构：will be doing。

时间：

  1. 将来将正在做的事情（疑问句时，多个助动词，将第一个助动词提到主语之前）
  2. 表示将要做的事，类似于一般将来，强调一个计划的安排性
  3. 可预见性的将来，询问别人的计划

被动语态

应用场合：不知道主语是谁；没必要表达主语是谁（语法成分重要的东西在前面：The temple is being repaired） ；强调动作的承受者。

A + do + B = B + be（体现时态，过去，现在，将来） + done（过去分词，表被动） + (by A)。被动语态：

  • 一般现在：am/is/are + done
  • 一般过去时：was/were + done
  • 现在进行：am/is/are + being + done
  • 过去进行：was/were + being + done
  • 现在完成：has/have + been + done
  • 过去完成：had + been + done
  • 一般将来：wiil + be + done
  • 过去将来：would + be + done

History is made by people.
The boy has been found by them.
The house is being repaired by him.
The bridge was built (by them) in 1980.

将来完成时

  • 使用场合：从现在看，截止到将来某时将已经完成的动作。
  • 谓语动词：will have done。

By the end of next month( by 表示截止 ), they will have been( 谓语动词 ) married ( 表示结婚的状态，结婚了的) for thirty years.
By the end of last month, they had been married for 30 years.
We will have finished the book by the end of this month.

现在完成进行时

过去一个动作发生，一直持续到现在，而且有可能持续下去。动作可能没有完成，如果完成也是刚刚完成。

subject + have /has been + doing

It has been raining for three days and it is still raining.
I have been waiting for an hour but she hasn’t come.
He has been running after her 8 years. （run after 追求）
I have been looking for you.

与现在完成时的区别：

  1. 现在完成时强调对现在的影响，进行时强调动作的持续；
  2. 不用于进行时态的动词不能用于现在完成进行时：
    I have known him for 8 years. 对
    I have been knowing him for 8 years. 错
  3. 肯定句中，表示一段时间的状语连用，时态可以互换；
    We’ve lived here for 2 years.
    We’ve been living here for 2 years.
  4. 不与时间状语连用是意义不同；
    I have been waiting for you. （还在等）
    I have waited for you. （已经走了）

过去完成进行时

subject + had + been doing，过去某时发生，一直延续到另一个过去的时刻。可能没有完成，完成也是刚刚完成。

I had been studying English before 2005.
Nobody knew what he had been doing all these years. （knew 体现过去的过去）

与过去完成时的区别：

  1. 过去完成：动作发生在过去；完成进行，强调过去某时的动作一直持续；
  2. 不用于进行时的动词不能用于过去完成进行时；
    He had loved her before they divorced. （正确）
    He had been loving her before they divorced. （错误）
  3. 肯定句中，与时间状语连用，可以互换，口语常用过去完成时。
    She had been waiting for him for 3 hours before he came back.
  4. 不与时间状语连用，意义不同。
    She had cleand the room so it was tidy. （打扫得很干净）
    She had been cleaning the room so we had to wait outside. （过去一直打扫，）

将来完成进行时

subject + will + have + been doing，某一个动作从某一个时刻开始，一直延续到将来的某一个时刻。

By this time next week, I will have been working for this company for 10 years.
Mr.Page will have been teaching for a total of forty years.
