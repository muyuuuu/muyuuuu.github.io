我不知道什么是爱。

两年前有了追人的冲动，也写过情书，表过白和约过会。因为不在一起上学而联系渐少，总会有些许的不甘，也会梦到再次见面的场景。不以自己单相思的角度出发，如果真的谈恋爱后该怎么做，矛盾了怎么办，会不会分手，毕业后该怎么办，等等等等，很多问题我都没想过。所以，我不知道什么是爱，也没有什么资格去谈恋爱。

2021-08-15

Debug 是需要经验的

记一次线程池的 debug：

  • 有的时候 debug 需要劳逸结合，不能着急。过段时间再看，结合理论看看代码，会有全新的想法，能成功找到 bug 所在
  • 有的时候 debug 需要锲而不舍，二分法逐步查找到错误所在，结合理论分析出为什么出错而后修改

debug 就像侦探小说一样，逐步分析问题所在，排除所有可能出错的语句，即使剩下的那条语句多么不可能出错，出错的也就是它了，然后一排查，还真是。

2021-02-23

编程不同阶段的感受

  • 初识，这些都好神奇，好玩
  • 刚入门，那些好玩的东西原来是这样，还有这么多神奇的东西
  • 学完基础课，很多事情原来没想象的那么简单，需要懂理论
  • 初次实践，到处是坑，涨了经验。也意识到之前写的程序真烂，能用的程序不等于好用的程序
  • 多次实践并回顾原理，意识到编程是件很危险的事

贯穿始终的感受：学的越多，知道的越少，也越能感觉到自己的渺茫，还差得很远。

2020年

2020-12-31

  • 大学毕业前，新年祝福无非是『生活开心，顺意』之类；列表里还在读大学的朋友，也无不在期待读研后的美好生活。
  • 唯独我们大学毕业的，发送的新年祝福和收到的新年祝福中，都有『早日暴富、早日经济自由、提前过上退休生活』的字眼。

看来不管是工作的还是读研的，大家都很累且很穷，唯有暴富才是硬道理，人间真实。今年我失去的东西够多了，也有没能和大学同学好好告别的一大遗憾。2021天天开心一切顺利是不可能的，只求被岁月温柔以待，有个好的归宿。

2020-08-30

还是要乘风破浪去黑暗历练一趟呢，什么时候才能被岁月温柔以待呢？

四年前考上二本的时候，一百个不甘心。本来以为保送后我的苦日子熬到头了，但一想到未来研究生的补助和生活，为数不多的快乐源泉也没了。一方面补助很少，工作的同学至少都能养活自己。我研究生也不求养活自己，本身家里都是普通的农民，收入很低，这把年纪还在大规模的消耗家里的钱财，实在让人没有继续读书和科研的热情，自己活着都需要这么啃老。生活方面，呵，研究生生活顺心是不可能的，只求活着。命运为何如此对我呢？艹。『民生多艰，爱而不得，家庭不幸，工作不幸』这都是什么人间疾苦，可还是要去磨练三年呢。年轻的时候总要去走走弯路的，希望这三年依旧年轻，依旧勿忘初心。看透了事物的本质，并没有勇气去热爱，我只是个凡人，不是个英雄。从小的学业和生活一直不顺心，起起落落落落落落的，那就希望以后运气好一点，顺点心吧。我没有骗人也不坑人，卖东西都不肯多要对面的钱，求上天给我一条温柔的出路吧。

我不敢说可以去理解任何人，因为我们无法感同身受，每个人的观念不同经历不同，导致了不可能去换位思考。如果可以换位思考，这世间怎么还会有抑郁症？

也不要去站在自己的角度或某个高度去帮助、批评他人，他知道自己错了会改，只是某种情绪、事件控制了他使他不能自拔(如恋爱中的悲伤总是难免的，如果他想消极一阵就随他去呗，劝有什么用呢？无法感同身受，换位思考也没用，自己的观点强加别人身上也没有意义)；如果他连错都不知道，那么没必要交这个朋友。

2019年

2019-12-29

见贤思齐焉，见不贤而内自省也。

这句话说的不是没有道理，前半句的意思是向优秀的人学习；后半句新的领悟是，当别人说出你的不足之处时，要冷静思考自己有没有这样的过错，有则表示感谢并改之，无则加勉。而不是在别人刚说完的时候就想出一堆理由去反驳他的观点，证明自己观点的正确性，这样只会像个泼妇，没有意义也没有意思。

后半句更新的领悟是，当指出别人的错误时，并不是自己有优越感。当别人指出我们的错误时，而不是不耐烦的反骂回去。无论对方语气如何，首先该知道对方没有这样的义务和责任来指出我们的错误。指出错误的人没有优越感，但无知且傲慢会有优越感。可能是网络快餐时代带来的浮躁吧，个别学生的脾气都大的很。

2019-10-15

学习是自己的事情，不能用任何结果来衡量。

保研后的真实经历：有人说，你已经保研了，为啥还在学习而不是去玩，考试分数对你来说无所谓了啊。

也许和他们不是一类人吧，尽管他们学历比我高出身比我好。学习是一种态度，是一种习惯，自己凭兴趣选的专业课，是热爱这个行业，不是为了考试，也不是为了保研。同样，考上初中、高中、大学，我也并不认为这是老师在开学时经常说的新起点，顺利考上大学不是某个阶段的起点也不是某个阶段的重点，同样保研不是大学的终点，也不是研究生的起点，而是中间的一个过程。还是要该干啥干啥，走在其中任何一个过程中，要把自己该做的做好。不以物喜，不以己悲，松弛有度方能收放自如。

五十年代有个叫皮特•西格的音乐家，写了一本教材要推广班卓琴，主动放弃了所有版权，任何人任何机构都随便印，只为了热爱。

无独有偶，电影护垫侠的主人公穆鲁戈南森，他说要用自己的发明造福妇女，拒绝了自己的机器注册专利。自己的东西可以随便山寨，只希望能造出物美价廉的卫生巾方便妇女，减少疾病。也是全球最有影响力的100人。

什么叫胸怀。这才是真正的热爱啊。

越厉害的人越谦虚，越自以为是的人却越态度傲慢，越爱搞所谓的版权夜郎自大。
