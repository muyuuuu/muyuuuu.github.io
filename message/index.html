2021-08-15

Debug 是需要经验的

记一次线程池的 debug：

  • 有的时候 debug 需要劳逸结合，不能着急。过段时间再看，结合理论看看代码，会有全新的想法，能成功找到 bug 所在
  • 有的时候 debug 需要锲而不舍，二分法逐步查找到错误所在，结合理论分析出为什么出错而后修改

debug 就像侦探小说一样，逐步分析问题所在，排除所有可能出错的语句，即使剩下的那条语句多么不可能出错，出错的也就是它了，然后一排查，还真是。

2021-02-23

编程不同阶段的感受

  • 初识，这些都好神奇，好玩
  • 刚入门，那些好玩的东西原来是这样，还有这么多神奇的东西
  • 学完基础课，很多事情原来没想象的那么简单，需要懂理论
  • 初次实践，到处是坑，涨了经验。也意识到之前写的程序真烂，能用的程序不等于好用的程序
  • 多次实践并回顾原理，意识到编程是件很危险的事

贯穿始终的感受：学的越多，知道的越少，也越能感觉到自己的渺茫，还差得很远。

哪一瞬间感觉自己老了？

周五坐公交出去玩，看见几个孩子穿着校服。阳关洒在他们脸上，一脸的青春稚嫩，眼睛有那种坚毅且清澈的光，这种光在成年人的眼睛里很难看到。永远会被少年感戳中，永远喜欢浪漫、不羁还带一些莽撞的少年气息。

2020-06-30

2016.9.12-2020.6.30，在这里上车，用四个小时告别了大学四年。

2019年最后一学期考完，2020.1.3 回家，2020.1.25 疫情爆发。疫情，大四下半年不让返校，最后只留一天回学校收拾东西；上午十一点到的，下午三点走的，用四个小时告别了大学四年。

坐在回去的车上，忽然发现很多人都没来得及见最后一面，或者说，也许只是一起吃了个饭说了声再见，不曾想某天的某个习以为常的时刻已经是人间的最后一面。很意外，很感慨，很无情。

我很想年轻两年在和你们一起度过一次大学，有很多珍贵的回忆，可是岁月不饶人啊。哪怕是深夜在机房指着对方鼻子骂街吵架，哪怕是炎热的夏天下课了去买瓶冰水，去操场跑步，吹吹海风……大学是我人生中最美好的回忆之一，匆忙的毕业没见同学最后一面也是人生最大的遗憾。

2020-02-21

不想换位思考

我不敢说可以去理解任何人，因为我们无法感同身受，每个人的观念不同经历不同，导致了不可能去换位思考。如果可以换位思考，这世间怎么还会有抑郁症？

也不要去站在自己的角度或某个高度去帮助、批评他人，他知道自己错了会改，只是某种情绪、事件使他不能自拔(如恋爱中的悲伤总是难免的，如果他想消极一阵就随他去呗，劝有什么用呢？无法感同身受，换位思考也没用，自己的观点强加别人身上也没有意义)；喜欢站在道德高点指责他人的人很烦。

见贤思齐焉，见不贤而内自省也。

在别人刚说完的时候就想出一堆理由去反驳他的观点，证明自己观点的正确性，这样只会像个泼妇，没有意义也没有意思。

后半句更新的领悟是，当指出别人的错误时，并不是自己有优越感。当别人指出我们的错误时，而不是不耐烦的反骂回去。无论对方语气如何，首先该知道对方没有这样的义务和责任来指出我们的错误。指出错误的人没有优越感，但无知且傲慢会有优越感。可能是网络快餐时代带来的浮躁吧，个别学生的脾气都大的很。

2019-10-15

学习是自己的事情，不能用任何结果来衡量。

保研后的真实经历：有人说，你已经保研了，为啥还在学习而不是去玩，考试分数对你来说无所谓了啊。学习是一种态度，是一种习惯，自己凭兴趣选的专业课，是热爱这个行业，不是为了考试，也不是为了保研。

同样，考上初中、高中、大学，我也并不认为这是老师在开学时经常说的新起点，顺利考上大学不是某个阶段的起点也不是某个阶段的重点，同样保研不是大学的终点，也不是研究生的起点，而是中间的一个过程。还是要该干啥干啥，走在其中任何一个过程中，要把自己该做的做好。
