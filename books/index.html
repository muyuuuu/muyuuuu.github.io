书籍推荐

回想我的大学，看了很多有价值的书，很感谢这些书及其作者。

所以我写了这样一个书籍推荐列表，这些书籍都是我都认真看过并觉得值得推荐的，同理，我读过但写得不好的书，我并不推荐也不会放在这个页面。希望能对你们有一些帮助，也以次记录和激励自己的读书生活。（虽然可能并无卵用）

在这个分类成立之初，当初很兴奋想弄好一点，结果可能因一时兴奋有很多书籍解读过度，这样并不好，如果看不顺眼请来信声讨。其实也无所谓，每个人的见解不同，视野不同，看到的内容自然也不相同，通过读书达到放松、休闲甚至获取精神食粮的目的即可，总比看那些抖音来的更有收获。

以下书目都是我的大学及其以后的生活中所读的书，中学读的就不放上来了，毕竟中学没读过几本。

大学所读

编程类

Python

书籍名称 描述
嵩天，礼欣，黄天羽. Python语言程序设计基础 第2版. 对我来说是入门的读物，了解python的简单特性和入门技巧，想要熟练还要多加练习。（仅通过此书很难精通）
Python Cookbook 不适合初学者读，适合有一定编程经验深入了解。思维方式和编程习惯都不错，实用性也可以，很多角落都有涉及。
PyQt5 快速开发与实战 对于初学者，几乎是目前最好的选择了。对于我这种有C#基础的，还行。解决了课设问题还有优研的答辩。

C语言

书籍名称 描述
C程序设计语言 入门经典，比谭浩强的更有意义。

文学类

偏悲剧

书籍名称 描述
挪威的森林 而长野的天，依旧那么暖，风吹起了从前。
了不起的盖茨比 于是我们奋力前进，却如同逆水行舟，注定要不停的退回过去。
围城 外面的人想进去，围在里面的人想出来

书籍名称 描述
麦田守望者 全书脏话连篇，至今没读懂

温情结尾

书籍名称 描述
简爱 在各种磨难中不断追求自由与尊严，坚持自我，最终获得幸福的故事，但绝不鸡汤。
假如给我三天光明 不可多得的传世经典，温暖清澄。
一个人的朝圣 温暖又孤独的旅行，一个退休老人为探望病危友人而独自踏上漫长旅程的故事。

书籍名称 描述
摆渡人(上下两本) I exist because you need me，因为你需要，所以我在这里，一直在这里。
追风筝的人 为你，千千万万遍。也许每个人心中都有一个风筝，无论它意味着什么，让我们勇敢的去追。一个不可多得的好故事。

励志类

书籍名称 描述
激流男孩 九个来自美国西部劳工家庭的大男孩，在1936年的奥运会上向全世界展示了什么是真正的勇气。
奔腾年代 海洋饼干不仅改变了他和他主人的命运，更激励了整个国家。

闲暇读物

书籍名称 描述
俗世奇人 泥人张，刷子李他们的故事。某次回学校的高铁上看完的。
莫泊桑短篇小说选 还记得羊脂球，变色龙这些幽默讽刺的故事吗，故事没有连贯性，四五页左右，读起来也不费劲。

数学类

书籍名称 描述
数学之美 强力推荐，无论你是数学建模还是AI入门。
统计学习方法 推导过程极为详细，有例子便不难看懂。

结语

可能还有很多看了没放上来，等哪天想起来了在这里补充，也欢迎有想安利给我书看的朋友在下方留言。
