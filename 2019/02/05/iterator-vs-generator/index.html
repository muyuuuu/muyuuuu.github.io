Iterator

Brief Introduction

Iteraote is primay method and object to type pythonic code, is also the way to make python stronger, which from the first element in set visit all of the elements forward not back and can remember position in visiting. Iterator has two basic methods iter() and next().

Iterator protocol:

  • Object with next() method, returns next element or throw an exception to stop iteration.
  • Object with iterator protocol is iterable object.
  • Iterable object realize iterator protocol, python built-in tools such as for, min, max and sum using iterator protocol to visit object.

Generator

Brief Introduction

Main advantage: return results when needs not everytime.

Generator can created by generation function and generation expression.

  • By function: with yield function, generator returns an iterator for program iteration. When generator runs will stop, save executing information, return yield values and remember position for next() method next run where has yield. Transfer a generator function will return an iterator object. Have a try to realize the Fibonacci sequence by using it.
