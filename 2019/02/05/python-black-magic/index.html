在numpy广播机制的作用下，应该避免去写for循环的程序，显得不够pythonic而且浪费时间和计算力。

可以发现，在对矩阵和一个元素求和时，numpy会对一个元素进行复制填充，使得一个元素的维度和矩阵维度相同，下面的例子中a+10中的10被填充为[[10,10,10]]。
