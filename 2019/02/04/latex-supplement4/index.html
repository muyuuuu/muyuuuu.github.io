LaTeX补充:常用实惠小工具

发表于 2019-02-04 更新于 2019-03-15 阅读次数：
本文字数： 715 阅读时长 ≈ 1 分钟

写 LaTeX 的时候排版表格是不是有想杀人的冲动？Ctex 发行版问题太多？数学符号不会写？参考文献管理太复杂？这里给出常用的实惠小工具，愉快的书写LaTeX 代码。

快速绘制表格：
http://excel2latex.com/

LaTeX 中数学公式转图片：
https://viereck.ch/latex-to-svg/

LaTeX 中书写不认识的数学符号：
http://detexify.kirelabs.org/classify.html?tdsourcetag=s_pctim_aiomsg

在线编译LaTeX：
https://www.overleaf.com/

好用的编辑器？sublime，vscode，notepad++，看个人口味。如果是vim大佬当然不在乎这些。

带相关开发环境的编辑器？texstduio（相当不错），texmaker（用过一次，功能好弱鸡），winedit（垃圾中的战斗机，latex都不收费，一个破编辑器还好意思收费）

刺激的发行版（编译器）？我个人是使用texlive的，不用考虑自己安装宏包的麻烦。前期安装复杂，后期操作绝对省心。下载地址：https://mirrors.tuna.tsinghua.edu.cn/

似乎看到了编程的两件套，编译器加编辑器或者IDE。

miktex只是听说过，会及时安装需要的包，前期省事可能后期麻烦。如果想学习，给个传送门：
https://camuseblog.top/2019-03-02-/MiKTeX/

miktex和texlive二选一即可，但是不要听任何老师和学长学姐的建议去安装Ctex，此发行版已经过时，近10年没人维护，到处是BUG，极其难用，正常代码都运行不了。
